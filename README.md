# Brighton & Hove — 5-Star Hygiene Guide

An interactive map and ranked list of restaurants, cafés and pubs in Brighton & Hove
holding a 5-star Food Standards Agency hygiene rating.

**View it here:** https://freemans13.github.io/hove-hygiene-guide/

Works on phones as well as desktop — the map sits above the list on a small screen.

## What's in here

- `index.html` — the guide itself. A single self-contained page; the only thing it
  pulls from the internet is the Leaflet mapping library, so it needs a connection
  but nothing else.
- `hove_5star_hygiene_ranking.pdf` — the same ranking as a printable document.

Ratings come from the Food Standards Agency's published data.
